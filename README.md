# Password Audit Toolkit

A Python-based cybersecurity toolkit for password analysis and cracking.

## Features
- Dictionary Generator
- Hash Generator
- Brute Force Engine
- John the Ripper Integration
- Password Strength Analyzer
- Report Logging System

## Usage

### Interactive Mode
python3 main.py

### CLI Mode
python3 main.py --hash admin123 --algo sha256

## Tools Used
- Python
- John the Ripper
